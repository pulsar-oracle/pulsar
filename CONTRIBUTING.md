# Contributing to Pulsar

Thank you for your interest in contributing!

## How to Contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feat/your-feature`
3. Commit your changes: `git commit -m "feat: your feature"`
4. Push and open a Pull Request

## Areas to Contribute

- New data feed adapters in `feeder/`
- On-chain aggregation logic in `contracts/`
- Tests and documentation
- Bug fixes and performance improvements

## Code Style

- Rust: run `cargo fmt` and `cargo clippy` before committing
- TypeScript: run `npm run lint` before committing

## Issues

Check open issues for tasks labeled `good first issue` or `help wanted`.
